# Provide the code that would allow inbound calls to one of our vitelity DID’s to take the inbound call and conversate intelligently backfilling data from the call transcription into our CRM / LMS and appointments scheduled in automated conversional fashion using our TELVERGENCE.com integration

Below is a practical **reference implementation** for an inbound-call pipeline that answers a Vitelity DID, records/transcribes the call, extracts structured data, writes it into your CRM/LMS, and creates an appointment in an automated conversational flow. The exact telephony hooks will depend on how your Telvergence layer exposes webhooks and media events, but the CRM transcript/association pattern is consistent with standard call-recording workflows.[^4_1][^4_2][^4_3]

## What it does

- Accepts the inbound DID call.
- Starts recording.
- Waits for call completion and transcription webhook.
- Uses an LLM to turn the transcript into structured fields.
- Backfills CRM/LMS records.
- Creates or updates an appointment.
- Sends SMS confirmation and internal alerts.


## Example architecture

1. **Inbound call hits your Telvergence/Telnyx-style webhook**.
2. **Your app answers and records the call**.
3. **Recording completes, transcription webhook fires**.
4. **Your app extracts lead intent, address, urgency, service type, and preferred callback window**.
5. **Your app writes the transcript and structured summary to CRM/LMS**.
6. **Your app books the appointment and sends SMS confirmation**.[^4_3][^4_4][^4_1]

## Python FastAPI example

```python
# app.py
import os
import json
import httpx
from fastapi import FastAPI, Request, BackgroundTasks
from pydantic import BaseModel
from openai import OpenAI

app = FastAPI()
client = OpenAI(api_key=os.environ["OPENAI_API_KEY"])

CRM_BASE = os.environ["CRM_BASE_URL"]
CRM_TOKEN = os.environ["CRM_API_TOKEN"]
LMS_BASE = os.environ.get("LMS_BASE_URL")
LMS_TOKEN = os.environ.get("LMS_API_TOKEN")
TELVERGENCE_BASE = os.environ.get("TELVERGENCE_BASE_URL")
TELVERGENCE_TOKEN = os.environ.get("TELVERGENCE_API_TOKEN")

class TranscriptPayload(BaseModel):
    call_id: str
    recording_url: str | None = None
    transcript: str
    from_number: str | None = None
    to_number: str | None = None
    duration_seconds: int | None = None

def llm_extract(transcript: str):
    prompt = f"""
Extract structured lead and appointment data from this call transcript.

Return strict JSON with:
{
  "customer_name": null,
  "phone": null,
  "email": null,
  "address": null,
  "city": null,
  "service_type": null,
  "urgency": null,
  "preferred_contact_method": null,
  "preferred_callback_window": null,
  "appointment_requested": false,
  "appointment_notes": null,
  "lead_score": 0,
  "summary": null
}

Transcript:
{transcript}
"""
    resp = client.responses.create(
        model="gpt-4.1-mini",
        input=prompt
    )
    text = resp.output_text
    return json.loads(text)

async def crm_upsert_lead(data, raw):
    payload = {
        "external_call_id": raw.call_id,
        "phone": raw.from_number,
        "name": data.get("customer_name"),
        "email": data.get("email"),
        "address": data.get("address"),
        "city": data.get("city"),
        "service_type": data.get("service_type"),
        "urgency": data.get("urgency"),
        "preferred_contact_method": data.get("preferred_contact_method"),
        "preferred_callback_window": data.get("preferred_callback_window"),
        "lead_score": data.get("lead_score", 0),
        "summary": data.get("summary"),
        "recording_url": raw.recording_url,
        "transcript": raw.transcript,
    }
    async with httpx.AsyncClient(timeout=30) as http:
        r = await http.post(
            f"{CRM_BASE}/leads/upsert",
            headers={"Authorization": f"Bearer {CRM_TOKEN}"},
            json=payload,
        )
        r.raise_for_status()
        return r.json()

async def create_appointment_if_needed(data, raw):
    if not data.get("appointment_requested"):
        return None

    payload = {
        "external_call_id": raw.call_id,
        "customer_name": data.get("customer_name"),
        "phone": raw.from_number,
        "address": data.get("address"),
        "service_type": data.get("service_type"),
        "window": data.get("preferred_callback_window"),
        "notes": data.get("appointment_notes"),
    }
    async with httpx.AsyncClient(timeout=30) as http:
        r = await http.post(
            f"{CRM_BASE}/appointments",
            headers={"Authorization": f"Bearer {CRM_TOKEN}"},
            json=payload,
        )
        r.raise_for_status()
        return r.json()

async def write_to_lms(data, raw, crm_result):
    if not LMS_BASE:
        return None
    payload = {
        "call_id": raw.call_id,
        "lead": data,
        "crm_result": crm_result,
        "transcript": raw.transcript,
    }
    async with httpx.AsyncClient(timeout=30) as http:
        r = await http.post(
            f"{LMS_BASE}/call-intake",
            headers={"Authorization": f"Bearer {LMS_TOKEN}"},
            json=payload,
        )
        r.raise_for_status()
        return r.json()

async def send_sms(to_number, body):
    if not TELVERGENCE_BASE:
        return None
    payload = {
        "to": to_number,
        "body": body,
    }
    async with httpx.AsyncClient(timeout=30) as http:
        r = await http.post(
            f"{TELVERGENCE_BASE}/sms/send",
            headers={"Authorization": f"Bearer {TELVERGENCE_TOKEN}"},
            json=payload,
        )
        r.raise_for_status()
        return r.json()

async def process_transcript(raw: TranscriptPayload):
    extracted = llm_extract(raw.transcript)
    crm_result = await crm_upsert_lead(extracted, raw)
    appt_result = await create_appointment_if_needed(extracted, raw)
    await write_to_lms(extracted, raw, crm_result)

    if raw.from_number:
        confirm = f"Thanks — we received your request for {extracted.get('service_type') or 'service'} and our team is reviewing dispatch and scheduling now."
        await send_sms(raw.from_number, confirm)

    return {"crm": crm_result, "appointment": appt_result, "extracted": extracted}

@app.post("/webhooks/inbound-call")
async def inbound_call(request: Request):
    payload = await request.json()
    return {
        "action": "answer_and_record",
        "record": True,
        "call_id": payload.get("call_id"),
        "next": "/webhooks/call-complete"
    }

@app.post("/webhooks/call-complete")
async def call_complete(request: Request, background_tasks: BackgroundTasks):
    payload = await request.json()
    raw = TranscriptPayload(**payload)
    background_tasks.add_task(process_transcript, raw)
    return {"status": "queued"}

@app.post("/webhooks/transcription")
async def transcription_webhook(request: Request, background_tasks: BackgroundTasks):
    payload = await request.json()
    raw = TranscriptPayload(
        call_id=payload["call_id"],
        recording_url=payload.get("recording_url"),
        transcript=payload["transcript"],
        from_number=payload.get("from_number"),
        to_number=payload.get("to_number"),
        duration_seconds=payload.get("duration_seconds"),
    )
    background_tasks.add_task(process_transcript, raw)
    return {"status": "queued"}
```

# A more specific TELVERGENCE webhook adapter

Below is a **specific webhook adapter** pattern you can use for a Telvergence-style PBX/telephony layer that receives inbound call and SMS events, normalizes them, and forwards them into your transcription, CRM, LMS, and appointment pipeline. The shape is based on common PBX webhook behavior: inbound call events often arrive as URL-encoded payloads, while SMS events commonly arrive as JSON, with event IDs or call UUIDs used for deduplication.[^5_1][^5_2]

## Adapter design

The adapter should do four jobs: verify authenticity, normalize payloads, deduplicate events, and enqueue downstream work. For inbound calls, accept the initial webhook, create a normalized call object, and return HTTP 200 immediately so the telephony platform is not blocked.[^5_3][^5_1]

## Expected events

A practical adapter usually handles:

- `InboundCall` for call arrival.
- `call.answered` or equivalent answer-state events if your provider exposes them.
- `call.recording.saved` when the recording is ready.
- `call.transcription.saved` when transcript text is available.
- `inboundMessage` for SMS intake and conversational follow-up.[^5_4][^5_1]


## FastAPI adapter example

```python
# telvergence_adapter.py
import os
import json
import urllib.parse
import hmac
import hashlib
from typing import Any, Dict
from fastapi import FastAPI, Request, Header, HTTPException, BackgroundTasks

app = FastAPI()

TELVERGENCE_SECRET = os.environ.get("TELVERGENCE_SECRET", "")
EVENT_CACHE = set()

def verify_signature(raw_body: bytes, signature: str | None):
    if not TELVERGENCE_SECRET or not signature:
        return
    digest = hmac.new(
        TELVERGENCE_SECRET.encode(),
        raw_body,
        hashlib.sha256
    ).hexdigest()
    if not hmac.compare_digest(digest, signature):
        raise HTTPException(status_code=401, detail="Invalid signature")

def dedupe(event_id: str | None):
    if not event_id:
        return False
    if event_id in EVENT_CACHE:
        return True
    EVENT_CACHE.add(event_id)
    return False

def normalize_inbound_call(payload: Dict[str, Any]) -> Dict[str, Any]:
    return {
        "event_type": "inbound_call",
        "call_id": payload.get("callUuid") or payload.get("call_id"),
        "did": payload.get("didNumber") or payload.get("toNumber"),
        "from_number": payload.get("callerIdNum") or payload.get("fromNumber"),
        "from_name": payload.get("callerIdName") or payload.get("fromName"),
        "tenant_id": payload.get("tenantId") or payload.get("tenantID"),
        "container_id": payload.get("containerId") or payload.get("containerID"),
        "raw": payload,
    }

def normalize_inbound_sms(payload: Dict[str, Any]) -> Dict[str, Any]:
    return {
        "event_type": "inbound_sms",
        "conversation_id": payload.get("conversationID"),
        "from_number": payload.get("fromNumber"),
        "to_number": payload.get("toNumber"),
        "message_text": payload.get("messageText"),
        "tenant_id": payload.get("tenantID"),
        "container_id": payload.get("containerID"),
        "raw": payload,
    }

async def handle_event(event: Dict[str, Any]):
    # enqueue your AI/CRM/LMS workflow here
    # e.g. send to a queue, worker, or webhook processor
    return

@app.post("/webhooks/telvergence/inbound-call")
async def inbound_call(
    request: Request,
    x_telvergence_signature: str | None = Header(default=None),
    background_tasks: BackgroundTasks = None,
):
    raw = await request.body()
    verify_signature(raw, x_telvergence_signature)

    form = urllib.parse.parse_qs(raw.decode())
    payload = {k: v[^5_0] if isinstance(v, list) and v else v for k, v in form.items()}
    event_id = payload.get("callUuid")
    if dedupe(event_id):
        return {"ok": True, "deduped": True}

    event = normalize_inbound_call(payload)
    if background_tasks:
        background_tasks.add_task(handle_event, event)
    return {"ok": True, "event": event["event_type"], "call_id": event["call_id"]}

@app.post("/webhooks/telvergence/inbound-sms")
async def inbound_sms(
    request: Request,
    x_telvergence_signature: str | None = Header(default=None),
    background_tasks: BackgroundTasks = None,
):
    raw = await request.body()
    verify_signature(raw, x_telvergence_signature)

    payload = json.loads(raw.decode())
    event_id = payload.get("messageID") or payload.get("conversationID")
    if dedupe(event_id):
        return {"ok": True, "deduped": True}

    event = normalize_inbound_sms(payload)
    if background_tasks:
        background_tasks.add_task(handle_event, event)
    return {"ok": True, "event": event["event_type"]}

@app.post("/webhooks/telvergence/transcription")
async def transcription_webhook(
    request: Request,
    x_telvergence_signature: str | None = Header(default=None),
    background_tasks: BackgroundTasks = None,
):
    raw = await request.body()
    verify_signature(raw, x_telvergence_signature)

    payload = json.loads(raw.decode())
    event_id = payload.get("transcriptionId") or payload.get("recordingId")
    if dedupe(event_id):
        return {"ok": True, "deduped": True}

    event = {
        "event_type": "transcript_ready",
        "call_id": payload.get("callUuid"),
        "recording_url": payload.get("recordingUrl"),
        "transcript": payload.get("transcript"),
        "duration_seconds": payload.get("durationSeconds"),
        "raw": payload,
    }
    if background_tasks:
        background_tasks.add_task(handle_event, event)
    return {"ok": True}
```

Use one worker that takes normalized events and routes them into transcription extraction, CRM backfill, LMS logging, and appointment scheduling.

async def process_event(event):
    if event["event_type"] == "inbound_call":
        # create placeholder lead + start recording workflow
        return

    if event["event_type"] == "transcript_ready":
        payload = {
            "call_id": event["call_id"],
            "transcript": event["transcript"],
            "recording_url": event.get("recording_url"),
        }
        async with httpx.AsyncClient(timeout=30) as http:

            if LMS_URL:
                await http.post(f"{LMS_URL}/calls", json=payload)
        return

    if event["event_type"] == "inbound_sms":
        async with httpx.AsyncClient(timeout=30) as http:
            await http.post(f"{CRM_URL}/sms/inbound", json=event)
        return

## Router architecture

1. **Webhook receiver** accepts inbound call, SMS, and transcription events.
2. **Normalizer** converts provider-specific payloads into one schema.
3. **Idempotency layer** checks whether the event was already processed.
4. **Classifier** determines whether the event is call-intake, transcript-ready, SMS, or appointment-ready.
5. **CRM worker** upserts lead/contact/deal fields.
6. **Scheduler worker** creates or updates appointments.
7. **LMS worker** logs the call, transcript, and disposition.
8. **Notification worker** sends SMS confirmation and internal alerts.

# next give you the exact worker queue schema and the database tables for storing calls, transcripts, leads, and appointment states.

Here’s a production-ready **queue schema + database model** for calls, transcripts, leads, and appointment states. It follows the webhook/idempotency pattern used in event-driven systems and keeps the raw event, processing status, and business records separate so retries do not corrupt data.[^7_1][^7_2][^7_3]

## Queue schema

Use one durable queue table plus a processing log. This lets the webhook receiver acknowledge fast, while a worker drains jobs safely in order of priority and retries failed items without duplicating business writes.[^7_4][^7_3]

```sql
CREATE TABLE job_queue (
  id BIGSERIAL PRIMARY KEY,
  event_id TEXT NOT NULL UNIQUE,
  tenant_id TEXT NOT NULL,
  event_type TEXT NOT NULL,
  aggregate_type TEXT NOT NULL,        -- call, transcript, lead, appointment
  aggregate_id TEXT,
  priority INT NOT NULL DEFAULT 100,
  status TEXT NOT NULL DEFAULT 'queued',  -- queued, processing, done, failed, dead
  attempts INT NOT NULL DEFAULT 0,
  max_attempts INT NOT NULL DEFAULT 8,
  available_at TIMESTAMPTZ NOT NULL DEFAULT NOW(),
  locked_at TIMESTAMPTZ,
  locked_by TEXT,
  payload JSONB NOT NULL,
  error_message TEXT,
  created_at TIMESTAMPTZ NOT NULL DEFAULT NOW(),
  updated_at TIMESTAMPTZ NOT NULL DEFAULT NOW()
);

CREATE INDEX idx_job_queue_status_available
  ON job_queue (status, available_at, priority);

CREATE INDEX idx_job_queue_tenant_event
  ON job_queue (tenant_id, event_type);
```


## Event log table

Keep a raw event ledger so you can replay or audit every webhook later. This is the safest place to enforce idempotency because the provider event ID is unique.[^7_2][^7_1]

```sql
CREATE TABLE webhook_events (
  id BIGSERIAL PRIMARY KEY,
  provider TEXT NOT NULL,
  provider_event_id TEXT NOT NULL UNIQUE,
  tenant_id TEXT NOT NULL,
  container_id TEXT,
  event_type TEXT NOT NULL,
  source_topic TEXT,
  signature_valid BOOLEAN NOT NULL DEFAULT FALSE,
  payload JSONB NOT NULL,
  received_at TIMESTAMPTZ NOT NULL DEFAULT NOW(),
  processed_at TIMESTAMPTZ,
  status TEXT NOT NULL DEFAULT 'received', -- received, queued, processed, failed, ignored
  error_message TEXT
);

CREATE INDEX idx_webhook_events_status_received
  ON webhook_events (status, received_at DESC);
```

  call_id TEXT NOT NULL REFERENCES calls(call_id) ON DELETE CASCADE,
  transcript_id TEXT UNIQUE,
  transcript_text TEXT NOT NULL,
  language_code TEXT,
  confidence NUMERIC(5,4),
  speaker_segments JSONB,

## Processing flow

1. Webhook arrives and is written to `webhook_events`.
2. A queue job is inserted into `job_queue` using the provider event ID as the unique key.
3. Worker locks the job, processes it, and writes to `calls`, `call_transcripts`, `leads`, or `appointments`.
4. If the transcript is ready, it updates the call row and creates or updates the lead.
5. If the caller asked for scheduling, it creates the appointment and logs history.
6. Job status moves to `done` or `failed` with retry metadata.[^7_3][^7_11][^7_1]

## Recommended status enums

- `job_queue.status`: `queued`, `processing`, `done`, `failed`, `dead`.
- `calls.status`: `open`, `in_progress`, `completed`, `missed`, `abandoned`.
- `leads.stage`: `new`, `contacted`, `qualified`, `booked`, `won`, `lost`.
- `appointments.status`: `requested`, `confirmed`, `rescheduled`, `completed`, `canceled`, `no_show`.


## Minimal worker claim query

```sql
WITH cte AS (
  SELECT id
  FROM job_queue
  WHERE status = 'queued'
    AND available_at <= NOW()
  ORDER BY priority ASC, created_at ASC
  FOR UPDATE SKIP LOCKED
  LIMIT 1
)
UPDATE job_queue j
SET status = 'processing',
    locked_at = NOW(),
    locked_by = 'worker-1',
    attempts = attempts + 1,
    updated_at = NOW()
FROM cte
WHERE j.id = cte.id
RETURNING j.*;
```

// database.js
const mongoose = require('mongoose');

// Create new workspace
async function createWorkspace(userData, didData, sipCredentials) {
    const workspace = new Workspace({
        workspace_id: `ws_${Math.random().toString(36).substring(7)}`,
        user_email: userData.email,
        user_name: userData.name,
        user_phone: userData.phone,
        company: userData.company,
        did_number: didData.did_number,
        did_id: didData.did_id,
        did_status: didData.status,
        did_capabilities: didData.capabilities,
        sip_username: sipCredentials.username,
        sip_password: sipCredentials.password,
        sip_domain: sipCredentials.domain,
        sip_proxy: sipCredentials.proxy,
        acrobits_app_id: `app_${Math.random().toString(36).substring(7)}`,
        acrobits_status: 'provisioned',
        tier: userData.tier,
        subscription_id: userData.subscription_id,
        subscription_status: 'active',
        payment_provider: userData.payment_provider,
        payment_id: userData.payment_id,
        storage_gb: getStorageByTier(userData.tier),
        max_dids: getMaxDidsByTier(userData.tier),
        features: getFeaturesByTier(userData.tier),
        activated_at: new Date(),
        metadata: {
            tier: userData.tier,
            company: userData.company,
            assigned_at: new Date().toISOString()
        }
    });
    
    await workspace.save();
    console.log('✅ Workspace created:', workspace.workspace_id);
    return workspace.workspace_id;
}

// Get storage by tier
function getStorageByTier(tier) {
    const storages = {
        starter: 5,
        professional: 25,
        enterprise: 100
    };
    return storages[tier] || 5;
}

// Get max DIDs by tier
function getMaxDidsByTier(tier) {
    const dids = {
        starter: 1,
        professional: 3,
        enterprise: 10
    };
    return dids[tier] || 1;
}

function getFeaturesByTier(tier) {
    const features = {
        starter: ['basic_collaboration'],
        professional: ['full_collaboration', 'photo_to_estimate'],
        enterprise: ['all_features', 'weather_dispatch', 'roofing_bundle']
    };
    return features[tier] || ['basic_collaboration'];
}

// Extract metadata from video file
async function extractMetadata(videoPath) {
    try {
        const metadata = {
            uploadedAt: new Date(),
            duration: 0,
            resolution: 'unknown',
            fps: 0
        };

        // Use ffmpeg to extract video metadata
        await ffmpeg(videoPath).getInfo((err, info) => {
            if (err) throw err;
            metadata.duration = info.duration;
            metadata.resolution = `${info.width}x${info.height}`;
            metadata.fps = info.fps;
        });

        console.log('✅ Metadata extracted:', metadata);
        return metadata;
    } catch (error) {
        console.error('❌ Metadata extraction error:', error);
        throw error;
    }
}

// Analyze video for roof damage
async function analyzeVideo(videoPath) {
    try {
        // Read video file
        const videoBuffer = fs.readFileSync(videoPath);

        // Send to Drone AI API for analysis
        const response = await axios.post(`${DRONE_AI_API_URL}/analyze`, {
            video: videoBuffer.toString('base64'),
            analysis_type: 'roof_damage'
        }, {
            headers: {
                'Authorization': `Bearer ${DRONE_AI_API_KEY}`,
                'Content-Type': 'application/json'
            }
        });

        const analysis = response.data.analysis;

        console.log('✅ Video analyzed:', analysis);
        return analysis;
    } catch (error) {

// Transcode video for web viewing
async function transcodeVideo(inputPath, outputPath) {
    try {
        await ffmpeg(inputPath)
            .outputOptions([
                '-vcodec libx264',
                '-acodec aac',
                '-preset fast',
                '-crf 23',
                '-vf scale=1280:720'
            ])
            .save(outputPath)
            .on('end', () => {
                console.log('✅ Video transcoded:', outputPath);
            });
    } catch (error) {
        console.error('❌ Video transcoding error:', error);
        throw error;
    }
}

async function generateThumbnail(videoPath, thumbnailPath) {
    try {
        // Extract frame at 5 seconds
        await ffmpeg(videoPath)
            .seekInput(5)
            .frames(1)
            .save(thumbnailPath);

        // Convert to sharp and optimize
        const thumbnail = await sharp(thumbnailPath)
            .resize(320, 180)
            .jpeg({ quality: 80 })
            .toFile(thumbnailPath);

        console.log('✅ Thumbnail generated:', thumbnailPath);
        return thumbnailPath;
    } catch (error) {
        console.error('❌ Thumbnail generation error:', error);

    }

async function generateQuote(analysis, metadata) {
    try {
        // Calculate material cost
        const materialCost = calculateMaterialCost(analysis.damagedArea, analysis.damageType);

        // Calculate labor cost
        const laborCost = calculateLaborCost(analysis.damagedArea, analysis.damageSeverity);

        // Calculate total cost
        const totalCost = materialCost + laborCost;

        // Estimate timeline
        const timeline = estimateTimeline(analysis.damagedArea, analysis.damageSeverity);

        // Determine warranty
        const warranty = determineWarranty(totalCost);

        // Estimate insurance coverage
        const insuranceCoverage = estimateInsuranceCoverage(analysis.damageType, totalCost);

        const quote = {
            quoteId: `quote_${Math.random().toString(36).substring(7)}`,
            materialCost,
            laborCost,
            totalCost,
            timeline,
            warranty,
            insuranceCoverage,
            generatedAt: new Date()
        };

        console.log('✅ Quote generated:', quote);
        return quote;
    } catch (error) {
        console.error('❌ Quote generation error:', error);
        throw error;
    }
}

function calculateMaterialCost(damagedArea, damageType) {
    const baseCostPerSqFt = {
        'shingle': 4.5,
        'metal': 9.0,
        'tile': 12.0,
        'slate': 15.0,
        'wood': 7.0
    };

    const costPerSqFt = baseCostPerSqFt[damageType] || 4.5;
    return Math.round(damagedArea * costPerSqFt * 100) / 100;
}

function calculateLaborCost(damagedArea, damageSeverity) {
    const baseLaborCostPerSqFt = 3.5;
    const severityMultiplier = 1 + (damageSeverity - 1) * 0.1;
    return Math.round(damagedArea * baseLaborCostPerSqFt * severityMultiplier * 100) / 100;
}

function estimateTimeline(damagedArea, damageSeverity) {
    const daysPerSqFt = 0.1;
    const severityDays = (damageSeverity - 1) * 2;
    const totalDays = Math.ceil(damagedArea * daysPerSqFt + severityDays);
    return `${totalDays} days`;
}

function determineWarranty(totalCost) {
    if (totalCost > 15000) {
        return '25-year warranty';
    } else if (totalCost > 10000) {
        return '20-year warranty';
    } else if (totalCost > 5000) {
        return '15-year warranty';
    } else {
        return '10-year warranty';
    }
}

function estimateInsuranceCoverage(damageType, totalCost) {
    const coveragePercentage = {
        'storm': 0.8,
        'water': 0.7,
        'fire': 0.9,
        'wear': 0.3,
        'other': 0.5
    };

    const percentage = coveragePercentage[damageType] || 0.5;
    return Math.round(totalCost * percentage * 100) / 100;
}

module.exports = {
    generateQuote

const FAA_API_URL = process.env.FAA_API_URL || 'https://api.faa.gov/v1';

// FAA Part 107 Compliance Checker
async function checkPart107Compliance(pilotId) {
    try {
        const response = await axios.get(`${FAA_API_URL}/pilots/${pilotId}/part107`, {
            headers: {
                'Authorization': `Bearer ${FAA_API_KEY}`,
                'Content-Type': 'application/json'
            }
        });

        const compliance = response.data;

        return {
            isCompliant: compliance.is_certified,
            pilotId: compliance.pilot_id,
            certificateNumber: compliance.certificate_number,
            certificateStatus: compliance.certificate_status,
            expirationDate: compliance.expiration_date,
            lastMedicalCheck: compliance.last_medical_check,
            restrictions: compliance.restrictions || []
        };
    } catch (error) {
        console.error('❌ Part 107 compliance check error:', error);

// Check Altitude & Speed Compliance
async function checkAltitudeSpeedCompliance(altitude, speed) {
    try {
        const maxAltitude = 400; // FAA max altitude in feet
        const maxSpeed = 100; // FAA max speed in mph

        const isAltitudeCompliant = altitude <= maxAltitude;
        const isSpeedCompliant = speed <= maxSpeed;

        return {
            isCompliant: isAltitudeCompliant && isSpeedCompliant,
            altitude: altitude,
            maxAltitude: maxAltitude,
            altitudeCompliant: isAltitudeCompliant,
            speed: speed,
            maxSpeed: maxSpeed,
            speedCompliant: isSpeedCompliant,
            warnings: []
        };

        if (!isAltitudeCompliant) {
            warnings.push(`Altitude ${altitude}ft exceeds FAA limit of ${maxAltitude}ft`);
        }
        if (!isSpeedCompliant) {
            warnings.push(`Speed ${speed}mph exceeds FAA limit of ${maxSpeed}mph`);
        }
    } catch (error) {
        console.error('❌ Altitude/speed compliance check error:', error);
        throw error;
    }

async function validateFlightPlan(flightPlan) {

        if (!flightPlan.latitude) {
            validation.isValid = false;

        if (!flightPlan.longitude) {

            validation.errors.push('Longitude is required');
        }
        if (!flightPlan.altitude) {
            validation.isValid = false;
            validation.errors.push('Altitude is required');
        }
        if (!flightPlan.start_time) {

            validation.errors.push('Start time is required');
        }
        if (!flightPlan.end_time) {

            validation.errors.push('End time is required');

# FAA Part 107 Compliance Guide for Telvergence Drone + Video-to-Roof-Quote

## Overview
All drone operations on the Telvergence platform must comply with **FAA Part 107** regulations for commercial drone operations.

## Required Compliance Checklist

### ✅ 1. Pilot Certification
- Pilot must hold **FAA Part 107 Remote Pilot Certificate**
- Certificate must be current and not expired
- Medical certificate (if required) must be valid
- Pilot must pass FAA recurrent knowledge test every 24 months

### ✅ 2. Drone Registration
- Drone must be registered with FAA (Registration number displayed on drone)
- Registration must be current (not expired)
- Drone weight must be under 55 lbs (25 kg)

### ✅ 3. Flight Operations
- **Maximum altitude: 400 feet** above ground level
- **Maximum speed: 100 mph** (87 knots)
- Must maintain **visual line-of-sight** with drone
- Must fly during **daylight or civil twilight** (or use anti-collision lights)
- Must yield right of way to **manned aircraft**
- Must **not fly from moving vehicle** (unless over sparsely populated area)

### ✅ 4. Airspace Authorization
- **LAANC authorization required** for controlled airspace (Class B, C, D, E)
- Submit flight plan at least 90 days before flight (recommended)
- Most authorizations granted within 24-48 hours

### ✅ 5. No-Fly Zones
- **Must not fly** in no-fly zones:
  - Airports (within 5 miles without authorization)
  - Military installations
  - National parks
  - Power plants
  - Washington DC SFRA
  - Temporary Flight Restrictions (TFR)

### ✅ 6. Insurance
- **Valid drone insurance required** (minimum $1M liability coverage recommended)
- Insurance must be current and not expired
- Insurance must cover commercial drone operations

### ✅ 7. Privacy Compliance
- **Provide privacy notice** to property owners before flight
- **Obtain flyover permission** if required by local law
- **Implement data protection policy** for drone footage
- **Limit footage collection** to purpose (roof inspection)
- **Secure stored footage** (encryption, access controls)
- **Delete footage** after retention period (recommended: 30 days)
