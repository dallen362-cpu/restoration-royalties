## Why some material is missing

This environment can read this repo, the GitHub repos (via the GitHub API), and public
web pages. It **cannot** access Claude.ai chats, Claude projects, or Perplexity
workspaces — so anything that lives only there (the dialer wiring, BOSS CRM, LMS, the
skills, and the Perplexity Max / Computer-mode setup output) has to be exported into the
repo before it can be consolidated. See `INTAKE-CHECKLIST.md`.
